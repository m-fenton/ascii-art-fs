ascii-art-fs

This task stems from the original ascii-art project with the addition of choosing a font for your string to be displayed in.
As a final argument in the command line you can choose the fonts "shadow', "standard" and "thinkertoy".

For example - 

go run . "your string" "thinkertoy"
